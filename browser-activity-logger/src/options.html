Browser Activity Logger

Settings for what gets recorded as you browse. Adjust these to match how much detail you want captured for later AI analysis.

What to log

Turn individual event types on or off. When an event type is off, those interactions are never recorded.

Where typed content is saved

By default only metadata about your interactions is stored (which field, what kind of element, page title). The actual content you type is saved only on the business domains you list here. Everywhere else, typed text is discarded and just the metadata is kept.

One domain per line. On these domains, the text you type may be saved alongside the event. Matched as a domain suffix (e.g. notion.so also covers www.notion.so).

Sensitive fields are never stored — passwords, auth tokens, credit-card numbers, and one-time passcodes (OTP) are always dropped regardless of any setting on this page, even for allowlisted domains.

Domains to exclude entirely

Nothing at all is logged on these domains — no metadata, no content, no events of any kind.

One domain per line. Matched as a domain suffix. Use this for banking, health, or any site you never want recorded.

Storage

Events are kept in a ring buffer. Once the cap is reached, the oldest events are dropped to make room for new ones.

A positive whole number (minimum 100). Higher values keep more history but use more local storage.
